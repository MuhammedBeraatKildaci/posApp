# MERN Stack POS App

## Technologies

- MongoDB
- Express
- React
- Node.js

## Features

- Add, update, delete categories
- Add, update, delete products
- Add, update, delete bills
- Add, update, delete products in a bill

## Installation

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
